HƯỚNG DẪN SỬ DỤNG CHƯƠNG TRÌNH QUẢN LÝ ĐIỂM SINH VIÊN

!Lưu ý: nên chạy trên VsCode để không xảy ra lỗi bất thường!

*Các thư viện cần tải xuống trước khi sử dụng:
- Pandas
+ Để tải xuống thư viện pandas thì chúng ta có thể sử dụng lệnh sau trong terminal:python -m pip install pandas
- Selenium
+ Để tải xuống thư viện selenium thì chúng ta có thể sử dụng lệnh sau trong terminal:python -m pip install selenium
- Undetected_chromedriver
+ Để tải xuống thư viện undetected_chromedriver thì chúng ta có thể sử dụng lệnh sau trong terminal:pip install undetected_chromedriver

*Cách sử dụng chương trình quản lý điểm sinh viên
-Chạy chương trình thông qua terminal bằng lệnh python Main.py
-Sau đó sử dụng thôi!

!Chúc bạn sử dụng vui vẻ!
